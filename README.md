# Calculator Android App (Java)

Android-порт калькулятора с Python/PyQt6 на Java для Android Studio.

## Функции

- ✅ Все базовые операции: +, −, ×, ÷
- ✅ Дополнительные: √, %, 1/x, ±
- ✅ Память: MC, MR, MS, M+, M-
- ✅ CE, C, ← (удаление)
- ✅ Отображение выражения и результата
- ✅ Секретный режим мини-игр (кнопка 🎮)
- ✅ Сохранение последнего результата
- ✅ Тёмная тема

## Установка

1. Откройте релизы проекта
2. Скачайте .apk файл
3. Установите скачанный файл
4. Готово!

## Требования

- Android 7.0+
