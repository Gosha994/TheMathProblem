# Calculator Android App (Java)

Android-порт калькулятора с Python/PyQt6 на Java для Android Studio.

## Функции

- ✅ Все базовые операции: +, −, ×, ÷
- ✅ Дополнительные: √, %, 1/x, ±
- ✅ Память: MC, MR, MS, M+, M-
- ✅ CE, C, ← (удаление)
- ✅ Отображение выражения и результата
- ✅ Секретный режим мини-игр (кнопка 🎮)
- ✅ Сохранение последнего результата
- ✅ Тёмная тема

## Установка

1. Откройте Android Studio
2. File → Open → выберите папку `CalculatorApp`
3. Дождитесь синхронизации Gradle
4. Run → Run 'app'

## Требования

- Android Studio Hedgehog или новее
- minSdk 24 (Android 7.0+)
- Java 8
